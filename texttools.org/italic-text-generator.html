About

This generator makes the text italic using characters from the Unicode block of Latin letters in italics and boldface. In typography, italic is a font style that slants slightly to the right side. Use this tool to apply italic font style in places where you normally can't use HTML tags or a rich text editor.

Bold-Italic Text

Enabling the “Bold” option will make the text both bold and italic at the same time. You can use it before or after the Unicode conversion.

Supported Characters

Only small a-z and capital letters A-Z are supported. Unsupported characters like numbers 0-9 are left unformatted since they currently have no equivalent in Unicode characters.
