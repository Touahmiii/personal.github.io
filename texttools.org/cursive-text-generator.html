About

This generator converts normal text into fancy cursive text using raw Unicode characters.

Cursive-Bold Text

Enabling the “Bold” option will make the text both cursive and bold at the same time. You can use it before or after the Unicode conversion.

Supported Characters

Only small a-z and capital letters A-Z are supported. Unsupported characters like 0-9 digits are left unformatted since they currently have no equivalent in Unicode characters.
