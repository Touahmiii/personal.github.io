About

Use this text tool to find unique lines in a plain text document and delete the copies to clean up your text. This tool can be useful to prevent repeating content in a list.

Trim

When “Trim” is enabled, leading and trailing spaces will be removed from each line.

Case Sensitive Match

When “Case Sensitive” is enabled, duplicates will be removed only if they match the letter case of the first occurrence. Disable it to perform a case insensitive (match case off) search.
